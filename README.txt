Project URL: https://roadmap.sh/projects/task-tracker

TaskTracker is a simple console application that allows users to manage their tasks efficiently. Users can add, view, and delete tasks from their task list.

## Features
- Add new tasks
- View all tasks
- Delete tasks by their index

## Prerequisites
- .NET 8 SDK

## Getting Started
1. Clone the repository or download the source code.
2. Open a terminal and navigate to the project directory.
3. Run the application using the command: `dotnet run`
